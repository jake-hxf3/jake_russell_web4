# Nested Grid & Flexbox
![A view of the car showroom](images/car-hero-xl.jpg)

This is a project about making a hero section for a Car Showroom using these techniques:
* Nested Grid
* FlexBox
* Responsive Image
* Order property


## Installation 💻
No installation required, simply download the zip file and extract.
## Usage
Open index.html in the browser of your choice and see the website.
